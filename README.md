# This is an AI agent project for [Boot.dev](https://boot.dev)

## Dependencies

> google-genai=1.12.1  
> python-dotenv=1.1.0

## Current Features
- Agent has access to the following capabilites
> - List contents of a directory in order to see what is contained
> - Read the contents of individual files, such as code or text
> - Write to a file as needed to make changes and fixes
> - Run a python script contained in the directory to evaluate it's output
